Project Tracker System Guide

System Overview

The Project Tracker Dashboard is a dynamic, filterable view powered by a central projects.json file, markdown logs, and a live dashboard UI.

  • /docs/logs/projects.json — Master index
  • /docs/logs/[project-name].md — Markdown logs
  • /docs/project-dashboard.html — Dashboard interface
  • /js/project-dashboard.js — Loader/renderer script
  • /css/components.css — Style system

How to Add a Project

  1. Create a log in /docs/logs/[project].md
  2. Convert to HTML (optional)
  3. Add to projects.json
  4. Reload the dashboard to verify

JSON Entry Format

{
  "title": "Nova Memory Dashboard",
  "html": "/docs/logs/nova-memory-dashboard.html",
  "tags": ["AI", "dashboard"],
  "status": "Active",
  "progress": 0.85,
  "lastUpdated": "2025-04-11",
  "latestLog": "Launched full Nova dashboard with modular awareness system."
}

Styling & Behavior

  • Uses .filter-pill for interactive filters
  • Grid layout via .grid-row
  • Progress bar only appears if defined
  • Status LEDs via emoji markers

Maintenance Tips

  • Use ISO dates YYYY-MM-DD
  • Keep tags consistent (no casing mismatches)
  • Keep logs synced (MD & HTML)
  • Short and clear latestLog notes
