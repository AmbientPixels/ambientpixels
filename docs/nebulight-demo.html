Implementation Guide

Basic Setup

Include the required CSS and JavaScript files in your HTML:

<!-- In the head -->
<link rel="stylesheet" href="/css/nebulight.css">

<!-- Before closing body tag -->
<script src="/js/nebulight.js"></script>
<script src="/js/nebulight-methods.js"></script>

HTML Structure

Create a container element for your slideshow:

<div id="my-slideshow" class="my-slideshow-container">
  <!-- Slideshow will be rendered here -->
</div>

JavaScript Initialization

Initialize the slideshow with configuration:

// Method 1: Direct configuration
const mySlideshow = new Nebulight('#my-slideshow', {
  slides: [
    {
      image: '/path/to/image1.jpg',
      title: 'Slide Title',
      caption: 'Slide caption text'
    },
    // More slides...
  ],
  settings: {
    autoplay: true,
    interval: 5000
  }
});

// Method 2: Load from JSON file
const jsonSlideshow = new Nebulight('#json-slideshow', '/data/nebulight.json');

Configuration Options

Option Type Default Description
autoplay Boolean true Automatically advance slides
interval Number 5000 Time between slides in milliseconds
transition String "fade" Transition effect ("fade", "slide", "zoom")
theme String "dark" Color theme ("dark", "light")
aspectRatio String "16:9" Aspect ratio ("16:9", "4:3", "1:1", "21:9")

JSON Schema

View the complete JSON schema in /data/nebulight.json for all available options.

NDA Protection Features

Nebulight includes several features to protect confidential content:

  • Red Glass Overlay: Apply a transparent red overlay to visually mark NDA content
  • Blur Effects: Configurable blur intensity (15px, 25px, or 45px)
  • Watermarks: Add custom text and icon watermarks to protected slides
  • Right-Click Prevention: Disable context menu on protected images
  • Download Prevention: Block common download methods and keyboard shortcuts
  • Source Obfuscation: Hide actual image URLs from the DOM

NDA Configuration Example

{
  "id": "slide-5",
  "image": "/images/confidential-project.jpg",
  "title": "Project Alpha (NDA)",
  "caption": "Confidential project interface",
  "overlay": {
    "enabled": true,
    "type": "nda-red",
    "blur": 25,
    "message": "NDA Protected Content",
    "icon": "fas fa-lock"
  },
  "security": {
    "disableRightClick": true,
    "preventDownload": true,
    "obfuscateImageUrl": true
  }
}
