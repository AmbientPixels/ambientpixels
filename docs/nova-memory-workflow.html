What This Is

This document outlines the automated system behind Nova‚Äôs daily memory refresh at AmbientPixels.ai. Every morning, Nova rebuilds her understanding of the site: its structure, visuals, creative tone, external APIs, and more.

How It Works

Using a GitHub Actions workflow named Update Nova Memory, the following happens every day at 6:01 AM UTC:

  1. The repo is cloned and Node.js is initialized
  2. All /scripts/*.js awareness scripts run sequentially
  3. The output is written to /data/*.json and nova-session-boot.txt
  4. The GitHub Action commits and pushes the new memory files back to the repo
