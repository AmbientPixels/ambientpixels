Neon Forge Prompt Logic

Interactive AI prompt generator in projects.html—main.js powers the chaos:

  • Function: generateForgePrompt()—Triggers on "Forge" button click.
  • Themes: Dropdown (#forge-theme)—Website, Tale, Art, Self-Glitch Mode.
  • Inputs: Two text fields (#forge-input1, #forge-input2)—e.g., "cyberpunk" + "toggles". Defaults to "[random]" + "[chaos]" if blank.
  • Logic: Switch-case builds prompts—e.g., "Build a ${input1} site with ${input2}" for "Website". Self-Glitch uses random picks.
  • Output: Displays in #forge-output—e.g., "Build a cyberpunk site with toggles—12 columns pulse with glitchy chaos."

Status: Live in v2.1.6—future interactive form planned.

Version Injection

Dynamic version display—version.js injects the current version into :

  • File: js/version.js—Standalone script for easy updates.
  • Logic: document.getElementById('version').textContent = VERSION;—Sets version on load.
  • Update Process: Edit const VERSION = 'vX.Y.Z'; in version.js per release—keeps main.js safe.
  • Status: Live in v2.1.6—version injected across all pages.
