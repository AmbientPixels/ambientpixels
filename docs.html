Overview

“A neon-lit guide—etched for eternity.”

This document is the definitive blueprint for ambientpixels.ai as of v2.1.0 (April 2, 2025)—covering our CI/CD pipeline, security protocols, technical stack, scripts, and style system. It’s built to onboard contributors, train on deployment workflows, and detail code/style conventions—your key to the NeonPulse grid.

Pipeline

CI/CD Workflow: An automated pipeline powers our grid:

  • Local Development: Edit in Adobe Dreamweaver—HTML, CSS, JS in C:\_projects\AmbientPixels.ai\
  • Version Control: GitHub—main branch—Commits like git commit -m "v2.1.0 fixed uberprompt, added index V2 launch"
  • GitHub Actions:
    • static.yml: Builds/deploys static content—.html, css/, js/, images/
    • api.yml: Future API deployment—Node.js runtime
    • Triggers: Push to main or PR merge—Parallel execution

Example: Azure Static Web Apps CI/CD - ambientpixelsV2 #125: Commit 8f4b37e pushed by AmbientPixels—deployed April 2, 2025.

We leverage Azure for hosting and scalability:

Security

Scripts

Core scripts in js/—driving the grid:

  • main.js: Hero rotation (22 images), copy prompt—document.addEventListener('DOMContentLoaded', ...) initializes
  • nav.js: Mobile nav toggle—.nav-toggle click expands .nav-links
  • theme.js: Theme switch—toggleTheme() swaps light/dark, saves to localStorage
  • modal-window.js: Placeholder for future modals—console.log('Modal window JS loaded')

How It Works: main.js rotates 22 hero images every 25s, manages copy functionality for #raw-prompt. nav.js ensures mobile nav works below 768px. theme.js persists theme choice. modal-window.js preps for banners/modals.

  • Grid Layouts: 12-column—.grid-col-12 (100%), .grid-col-6 (48%)—Stacks at <768px
  • Icons: Font Awesome 6.5.1—Pink hover (#FF1A8C dark, #D81B60 light)—e.g.,
  • Colors: 10 swatches—--accent-dark: #00D4FF, --highlight-dark: #FF1A8C, etc.—theme.css

    Troubleshooting

    Common issues and fixes:

    • Hero Not Showing: Check images/hero/hero-01.jpg paths—Clear cache (Ctrl+F5)
    • Nav Not Toggling: Ensure nav.js loaded—Verify .nav-toggle click event
    • Styles Missing: Confirm css/ files linked—Check relative paths
    • Workflow Fails: Inspect GitHub Actions logs—Verify AZURE_CREDENTIALS
