Style Guide

Typography

Inter font—crisp, futuristic—400/500 weights. Headlines glow white—text-shadow: 0 0 5px—body text softens to #D8E0E5 (dark) or #1A2A44 (light). Links hover cyan (#87ceeb)—transition: color 0.3s ease.

Colors

Dark: #071019 (bg), #D8E0E5 (text), #2A577A/#4B3E8F (accents). Light: #F5F7FA (bg), #1A2A44 (text), #3B82F6 (accent). Shadows deepen—rgba(0, 0, 0, 0.3)—for moodier tech vibes.
