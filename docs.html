Managing Banners

Control the EchoGrid pulse—add or remove banners with ease:

  • Location: Edit C:\ambientpixels\EchoGrid\js\banner.js.
  • Structure: Find the banners array—each entry is a banner with level (high/medium/low), icon (Font Awesome), and text.
  • Add a Banner:
    1. Open banner.js in VS Code.
    2. Append to banners: { level: 'medium', icon: 'fas fa-star', text: 'New Drop Incoming—Stay Tuned!' }.
    3. Save, test with Live Server—rotates every 5s.
  • Remove a Banner:
    1. Open banner.js in VS Code.
    2. Delete an entry (e.g., remove Kavinsky: { level: 'low', icon: 'fas fa-music', text: '...' }).
    3. Save, test—gone from rotation.
  • Tips: Keep text short (~15-20 words), use Font Awesome icons, hover to pause, click × to hide.

Image Guidelines

Keep the grid’s visuals neon-sharp—here’s how we size, name, and spark ‘em:

  • Sizes:
    • Heroes: 1920x1080px (16:9)—Full-screen chaos, 500px height in CSS. Used in index.html, skills.html.
    • Mini-Heroes: 960x540px (16:9)—Compact teasers, 200px height in CSS. Used in test.html, dev.html.
    • Cards: 600x150px (4:1)—Wide promo tiles, 150px height in CSS. Used in index.html.
  • Format: JPG default—small, fast (70-80 quality). PNG optional for transparency (e.g., glowing overlays).
  • Naming Convention: [type]-[2-digit-number]-[short-desc].[ext]
    • type: hero, mini-hero, card.
    • number: 01-99—sequential order.
    • short-desc: 1-2 words, lowercase, hyphenated—e.g., neon-city.
    • ext: .jpg or .png.
  • Locations:
    • images/hero/—Heroes (e.g., hero-01-neon-city.jpg).
    • images/mini-hero/—Mini-Heroes (e.g., mini-hero-01-cyber-alley.jpg).
    • images/cards/—Cards (e.g., card-07-streambeats.jpg).
  • MidJourney Prompt Examples:
    • Hero: /imagine prompt: A sprawling cyberpunk cityscape at night—towering neon skyscrapers pulse with electric blues and pinks, holographic billboards flicker, drones zip through misty air—hyper-detailed, cinematic lighting, dark futuristic chaos --v 5 --ar 16:9 --q 2 → hero-01-neon-city.jpg
    • Mini-Hero: /imagine prompt: A narrow neon-lit alley—flickering signs in red and green, wet pavement reflects glowing glyphs, shadowy figure in a trench coat—gritty, compact, cyberpunk edge --v 5 --ar 16:9 --q 2 → mini-hero-01-cyber-alley.jpg
    • Card: /imagine prompt: A wide glitch-art banner—distorted neon pixels in pink and teal, cyberpunk figure with glowing eyes, jagged text ‘DROP INCOMING’—chaotic promo vibe --v 5 --ar 4:1 --q 2 → card-08-glitch-art.jpg
  • Tips: Use JPG for speed—convert PNGs in Photoshop (Save for Web, 70-80 quality). Name consistently—e.g., card-09-synth-beat.jpg. Store in right folders—keeps the grid clean!
