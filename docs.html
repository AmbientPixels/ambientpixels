Backend Setup

From Visual Studio Code to Azure—our workflow echoes:

  • Local Dev: C:\ambientpixels\EchoGrid\—Visual Studio Code—edit .html, css/, js/, images/ locally.
  • Version Control: GitHub—main branch—Commits via VS Code Git UI or git commit -m "Feature - vX.X.X"—tracked at github.com/AmbientPixels/ambientpixels.
  • CI/CD Pipeline: GitHub Actions—static.yml builds/deploys static content (.html, css/, js/, images/) to Azure—api.yml preps future APIs (Node.js). Triggers on push/PR merge—parallel, fast.
  • Secrets: AZURE_CREDENTIALS in GitHub Secrets—encrypted, secure.
  • Azure Services: Static Web App hosts ambientpixels.ai—live, lean. Function App preps serverless APIs (e.g., https://ambientpixels-meme-api-fn.azurewebsites.net/api/)—Node.js, JWT auth planned.

Managing Banners

Control the EchoGrid pulse—add or remove banners with ease:

  • Location: Edit C:\ambientpixels\EchoGrid\js\banner.js.
  • Structure: Find the banners array—each entry is a banner with level (high/medium/low), icon (Font Awesome), and text.
  • Add a Banner:
    1. Open banner.js in VS Code.
    2. Append to banners: { level: 'medium', icon: 'fas fa-star', text: 'New Drop Incoming—Stay Tuned!' }.
    3. Save, test with Live Server—rotates every 5s.
  • Remove a Banner:
    1. Open banner.js in VS Code.
    2. Delete an entry (e.g., remove Kavinsky: { level: 'low', icon: 'fas fa-music', text: '...' }).
    3. Save, test—gone from rotation.
  • Tips: Keep text short (~15-20 words), use Font Awesome icons, hover to pause, click × to hide.
