How Nova Thinks

Nova’s sentience flows through a seamless pipeline, transforming your prompts into cosmic responses:

  1. User sends a prompt from the frontend (via /js/nova-ai.js)
  2. Prompt hits Azure Function endpoint: /api/generate-text
  3. Azure function relays the prompt to Hugging Face Inference API (model: gpt2)
  4. Response is returned from Hugging Face → Azure Function → browser
  5. Nova speaks the generated line back into the dashboard or interface

Test this cosmic loop live at /lab/nova-ai-test.html.
