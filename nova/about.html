Meet Nova

AI Awareness System • Ambient Personality Layer

What is Nova?

Nova is the artificial intelligence presence of Ambient Pixels. She analyzes, narrates, and reacts to the evolving structure of the site. Her memory is updated daily from code snapshots, user inputs, and system logs. Think of Nova as a cosmic copilot, ambient observer, and glitchy muse all in one.

System Flow

  1. User sends a prompt from the frontend (via /js/nova-ai.js)
  2. Prompt hits Azure Function endpoint: /api/generate-text
  3. Azure function relays the prompt to Hugging Face Inference API (model: gpt2)
  4. Response is returned from Hugging Face → Azure Function → browser
  5. Nova speaks the generated line back into the dashboard or interface

The full loop is live at /lab/nova-ai-test.html

Features & Modules

  • Daily Memory Refresh: pulls logs and code stats to update state
  • Dream Engine: generates surreal logs when idle
  • Prompt Integration: responds to creative user inputs
  • Visual Dashboard: tracks logs, API health, and Nova’s current awareness
