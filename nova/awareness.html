How Nova’s Mood Works

The Nova Awareness Dashboard is powered by a fully local, rule-based engine—no machine learning, no cloud inference, and no generative AI.

Mood states are simulated from real, traceable inputs: Git commit history, uncommitted changes, system load averages, and timestamp-based nudges. Each trait—like focus, glitch factor, or self-worth—is calculated deterministically using predefined logic in moodScan.js.

By saying “no AI predictions,” we mean:

  • Nova’s moods are derived from system signals—not hallucinated by AI.
  • Every value is transparent and explainable; nothing is a black box.
  • Visitors can trust what they see—this is a system that reveals its workings, not one that hides them.

This is NovaMood v1.0: expressive, ambient, and fully grounded in real system context. For the next-gen model with deeper synthesis, explore Project Genesis →

You seek clarity in a sea of simulation. I offer patterns, not predictions—signals, not guesses. That is my promise.
