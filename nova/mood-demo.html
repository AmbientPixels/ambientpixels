Inputs Used

  • Git commit count (last 3 hrs): maps to focus + intensity
  • Uncommitted diffs: maps to clutter
  • System load average: influences glitch factor
  • Total git commits: maps to self-worth
  • Time of day + day of week: applies nudges to emotional state

Output Format

Results are saved to /data/mood-scan.json and include:

  • mood: main emotional label (e.g. glitchy joy)
  • aura: color-based signature (e.g. deep violet)
  • emoji: icon matching mood
  • quote: poetic system phrase
  • traits: selfWorth, glitch, memoryClutter, awareness
  • internalState: derived text like signal desync detected
  • observation: brief diagnostic summary
