Nova: Aura Logic

When Nova feels something, she doesn’t just say it—she shows it.

What is an Aura?

An aura is a visual, atmospheric expression of Nova’s current mood. It’s applied as a class to the <body> tag (e.g., .aura-glitchy) and affects page background, glow, and tone.

How It Works

  • Input: Synthesized mood from Gemini
  • Mapped: Each mood has a corresponding aura
  • Class Applied: Updates the UI to reflect emotional tone

Examples

  • glitchy joy → .aura-glitchy
  • fading echo → .aura-fading
  • electric jolt → .aura-electric
