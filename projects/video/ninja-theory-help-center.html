Project Overview

This animated showcase introduces the redesigned Ninja Theory Help Center on support.Xbox.com. Crafted in collaboration with the Xbox Support content team, the video highlights key platform features for games like Hellblade and Senua's Saga: Hellblade II. All visuals were animated in After Effects, narrated by Azure AI voice with SSML enhancements, and rendered using the full Adobe production suite.

AI Production Pipeline

This project was brought to life using an advanced AI-driven production pipeline that enhanced creativity and efficiency at every stage. Here's a look at the tools and processes that powered this production:

Creative Concepting

Leveraged ChatGPT, Claude, and Gemini to brainstorm ideas, outline scripts, and generate wireframes or scene sketches, establishing a strong creative direction and visual tone from the start.

Project Management & Guidance

Utilized Nova, a custom GPT-based project guide, to track goals, manage timelines, and provide creative decision support throughout the production process.

Animation & Motion Design

Enhanced After Effects with AI tools like Runway, script-assist agents, and pre-trained motion models to automate transitions, generate keyframes, and align visuals to audio, maintaining consistent motion rhythm across scenes.

Asset Creation

Combined Adobe Firefly, Midjourney, and Photoshop plugins with custom AI assistants to generate textures, visual effects, and imagery, creating rich, layered environments.

Voice & Audio

Created Ava, a branded voice agent built with Azure Voice Studio, using SSML scripting and GPT-assisted prompts to control tone, pacing, and clarity for natural-sounding narration.

UI & Design Feedback

Deployed a review agent using ChatGPT Vision to analyze UI screenshots, providing automated feedback on clarity, spacing, contrast, and accessibility throughout the design process.

Script & On-Screen Text

Utilized agents like ChatGPT and Notion AI to co-develop scripts, adjust tone, and generate short-form copy for titles, subtitles, and screen overlays, ensuring viewer-friendly content aligned with brand voice.

AI Agents & Tools

Ava - Voice Agent

Custom AI voice assistant built with Azure Voice Studio, using SSML scripting for precise control over narration. Handles all voiceover work with natural intonation and brand-consistent delivery.

Vision - Design Agent

ChatGPT Vision-powered assistant that analyzes UI screenshots, providing real-time feedback on design elements, spacing, and accessibility compliance.

Runway ML

Integrated into After Effects for AI-assisted motion design, automating rotoscoping, and generating complex motion patterns while maintaining visual consistency.

ScriptFlow

Custom agent combining GPT-4 and Notion AI for script development, generating multiple versions of on-screen text and maintaining brand voice consistency.

Editorial Agent

AI-powered editorial assistant that refines scripts and generates on-screen text, ensuring consistency in tone, style, and brand voice while optimizing content for clarity and engagement.

Behind the Scenes: Technical Production

Animation Pipeline

  • Motion Design: Adobe After Effects CC 2023 with custom scripts for batch processing
  • AI-Assisted Workflow: Automated rotoscoping and motion tracking using Runway ML
  • Version Control: GitHub for project file management and collaboration
  • Design System: Custom AE template with responsive layouts for 16:9 and 9:16 outputs

Visual Design

  • Vector Assets: Adobe Illustrator CC with dynamic data linking
  • Color Grading: DaVinci Resolve for HDR color management
  • Effects: Red Giant Universe and Sapphire plugins for advanced visual treatments
  • Composition: 1920x1080 @ 30fps, sRGB color space

Audio Production

  • Voiceover: Azure AI Neural Voices with custom SSML tuning
  • Mixing: Adobe Audition with iZotope RX for noise reduction
  • Scoring: AI-generated music with AIVA, mixed in Logic Pro
  • Delivery: Stereo mix at -16 LUFS, 48kHz sample rate

Technical Highlights

Delivery & Encoding

Delivered final assets via shared internal SharePoint for review and provided full project zip files including source files, assets, and project files. Encoded in H.264 (MP4) and ProRes 422 HQ formats for optimal quality and compatibility.

Audio Production

Utilized Adobe Premiere Pro's seamless looping functionality to extend background music tracks while maintaining natural transitions and audio quality. Applied crossfades and EQ adjustments to ensure smooth, continuous playback.

AI Voice Generation

Created branded AI voiceovers using SSML (Speech Synthesis Markup Language) for precise control over pronunciation, emphasis, and pacing. Fine-tuned parameters including pitch, rate, and volume to match the desired brand voice characteristics.
