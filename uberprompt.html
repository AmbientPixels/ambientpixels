Project Overview

“A lightweight grid hums—future-proofed and neon-lit.”

Our mission—scope: v2.3;—is to forge a modular, cyberpunk-inspired site at ambientpixels.ai/test. Starting at v0.17.8+, it’s a scalable testbed for core features—portfolio-ready, CI/CD-aligned, balancing minimal flair with maximum potential. Crafted with PixelPulse & Grok on April 1, 2025, this codex is our eternal blueprint—every detail unfolds below in neon brilliance.

Top Frosty Menu

A sticky, frosted-glass navigation bar crowns the site—minimalist, icon-driven, and ever-present.

  • Logo: images/ambient-pixel-logo-rainbow.png—Left-aligned, 40px height, a rainbow glow symbolizing our cyberpunk ethos.
  • Links: Icon-only for sleekness—Home (), Test (), Projects (), Art (), Tools ()—portals to our grid.
  • Theme Toggle: Moon icon ()—Switches light/dark modes, stored in localStorage for persistence.

Responsive: Below 768px, a hamburger toggle () expands .nav-links into a full-width dropdown—20px padding, frosted backdrop. Style: Frosted effect via backdrop-filter: blur(5px), light/dark theme support—box-shadow: 0 0 15px rgba(0, 212, 255, 0.3) casts a cyan glow. Accessibility: ARIA labels (e.g., aria-label="Toggle Navigation") ensure WCAG compliance—low-risk, user-friendly.

Grid System

“Columns flex—content breathes.”

A modular 12-column grid forms the backbone, paired with a sleek footer—responsive, toggle-ready, and built for chaos.

  • Breakpoints: Mobile (<768px)—stacks to 100%; Tablet (768-1024px)—48% cols; Desktop (>1024px)—flexible 48% or 100%—fluid adaptation.
  • Toggleable Sections: Planned—Video Projects, Art Gallery, Client Work, Dev Tools—each a
    with child .content, hidden by default, toggled via data-toggle buttons—future interactivity hub.
  • Base Content: Title, subheading, quote, list, toggle button—lightweight, extensible, ready for neon flair.

Footer: Links—Contact (#contact), Copyright 2025 (), Back-to-Top (#). Social icons—DeviantArt, X, YouTube, Facebook—hover glows pink (text-shadow: 0 0 10px var(--highlight-dark)). Tagline: “Built in the dark, glowing by starlight”—our creed. Style: Clean, modern—light/dark themes, subtle shadows (box-shadow: 0 2px 5px rgba(0,0,0,0.1)). Accessibility: ARIA for toggles (aria-expanded="false")—low-risk, WCAG-ready.
